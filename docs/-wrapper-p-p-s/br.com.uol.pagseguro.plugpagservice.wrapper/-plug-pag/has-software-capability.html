Parameters

operation

Valor da funcionalidade a ser validada. Possíveis valores em: PlugPagCommand. É necesário usar o valor de cada operação, seguindo o exemplo: PlugPagCommand.OPERATION_ACTIVATE.value().

mode

Modo da funcionalidade a ser verificada, podendo ser: PlugPagCommand.OPERATION_MODE_REMOTECFG, em caso de configurações remotas e PlugPagCommand.OPERATION_NO_EXTRA_PARAMS em casos gerais. É necesário usar o valor de cada operação, seguindo o exemplo: PlugPagCommand.OPERATION_NO_EXTRA_PARAMS.value()
