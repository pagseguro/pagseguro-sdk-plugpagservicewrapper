getPreAutoList

abstract fun getPreAutoList(): PlugPagPreAutoQueryResult

Executa uma consulta de pré-autorização via cartão.

Return

Em caso de sucesso (PlugPagPreAutoQueryResult.result == PlugPag.RET_OK), o resultado da consulta contendo a lista de pré-autorizações (PlugPagPreAutoQueryResult.transactions) associadas ao cartão. Em caso de falha (PlugPagPreAutoQueryResult.result != PlugPag.RET_OK), o código (PlugPagPreAutoQueryResult.errorCode) e a mensagem de erro (PlugPagPreAutoQueryResult.message).
